## Normal Mode
- ``:q`` quit
- ``:w`` save
- ``:o + filename`` open a file
- ``x`` delete a character
- ``u`` undo
- ``r`` redo
- ``i`` shift to Insert Mode

## Insert Mode
- ``esc`` to Normal Mode
